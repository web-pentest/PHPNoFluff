# 🐘 PHPNoFluff

Просто PHP. Без воды.

## О проекте

Я PHP-юзер и разобрал весь синтаксис в двух файлах.  
Одному файлу верь как шпаргалке. Второй читай как учебник.

## 📁 Файлы

| Файл | Для кого | Что внутри |
|------|----------|------------|
| **LEARN.php** | Кто уже кодит ⚡ | Быстрый синтаксис, примеры, компактно |
| **TUTORIAL.php** | Кто учится с нуля 🎓 | 12 глав, пояснения, задания, безопасность |

## 📚 Что внутри

| Тема | Описание |
|------|----------|
| `$` Переменные | типы, область видимости, приведение |

| 📝 Строки | '', "", heredoc, nowdoc, функции |
| 🗂️ Массивы | индексные, ассоциативные, многомерные |
| 🔀 Условия | if, else, elseif, switch, тернарка |
| 🔄 Циклы | for, while, foreach, do-while |
| ⚙️ Функции | параметры, return, анонимные, стрелочные |
| 🏗️ ООП | классы, объекты, наследование, трейты |
| 🌍 Суперглобальные | $\_GET, $\_POST, $\_SESSION, $\_COOKIE |
| 🗄️ Базы данных | PDO, подготовленные запросы |
| 🛡️ Безопасность | XSS, SQL-инъекции, хеширование |
| ⚠️ Ошибки | try-catch, исключения |
